Options / Settings

LWB provides options through the toolbar, the menu "Options" or the key combination "Alt + s".

Truth tables show...

Usual truth tables show all possible combinations of valuations. With this setting you can choose to only show truth table rows, where the result is true; or false.

SAT solutions show...

When the SAT solver finds a valid solution for a formula, the valuations are presented in a table per default. You might also choose to show a table only of the variables that are true. The third option shows the complete result of the SAT solving like the first option, but as a formula.

MMP include path

Use this field to add directories, in which the program should search for M4 files. The paths can be absolute or relative, beginning from the applications root folder. Use semicolons to seperate paths.
